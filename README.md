# Timer (Frontend Prototype)

A tiny **no-dependency** **static UI prototype** for a timer (HTML/CSS only).  
It’s meant for layout/styling — it does **not** implement countdown logic.

## Run it

From the repo root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Files

- `index.html`: UI
- `styles.css`: styling
