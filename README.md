# Simple Timer App

A tiny **no-dependency** timer (HTML/CSS/JS) with start/pause/reset, presets, a progress bar, and keyboard shortcuts.

## Run it

From the repo root:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Files

- `index.html`: UI
- `styles.css`: styling
- `app.js`: timer logic
